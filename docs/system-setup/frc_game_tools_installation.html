FRC Game Tools Installation

Before we are able to program our robot, we need to install a software called FRC Game Tools

Package info

The FRC Game Tools installs LabVIEW, FRC Drive Station, and FRC roboRIO Image Tool

System requirements
  • Operating System: Windows 10 or Windows 11
  • Architecture: x86 (64-bit). ARM-based devices are not supported
